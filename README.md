## Yarn Commands

Install dependencies:

```sh
$ yarn install
```

Run build:

```sh
$ yarn build
```

Run tests:

```sh
$ yarn test
# or with file watcher
$ yarn test:watch
```

Run a TypeScript file:

```sh
$ yarn eval app/index.ts
```

Run the web application:

```sh
$ yarn web
```
then click on demo links.
